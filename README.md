# Springboot和Vue3图书管理系统蓝白前后端分离 java250

## 项目概述

本项目是一个基于Springboot和Vue3的图书管理系统，实现前后端分离的设计模式。系统支持Java源码并提供数据库文档，旨在为图书管理人员、教师和学生提供一个便捷、高效的管理平台。

## 技术栈

- 前端：Vue3
- 后端：Springboot
- 构建工具：Maven
- 数据库：MySQL
- 主要编程语言：Java、JavaScript

## 功能模块

### 一、主页
- 欢迎页：展示系统概览

### 二、图书管理
- 图书检索
- 图书借阅
- 图书归还
- 图书新增
- 图书修改
- 图书删除

### 三、用户管理
- 学生信息管理
- 教师信息管理
- 图书管理员信息管理

### 四、系统设置
- 权限配置
- 参数设置
- 系统日志

## 系统角色

- 学生：图书检索、借阅、归还
- 教师：图书检索、借阅、归还、管理个人信息
- 图书管理员：图书管理、用户管理、系统设置

## 运行环境

- 后端：JDK 1.8 或以上版本
- 前端：现代浏览器
- 数据库：MySQL 5.7 或以上版本

## 部署步骤

1. 安装并配置好MySQL数据库环境
2. 创建数据库，导入数据
3. 部署后端服务，配置application.properties文件
4. 部署前端服务，配置接口地址
5. 启动前后端服务
